# 🌦️ Weather App

A responsive Weather App built using **HTML, CSS, and JavaScript** that fetches real-time weather data using the **OpenWeatherMap API**.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature (°C)
- 💧 Humidity (%)
- 🌬️ Wind Speed (km/h)
- 🌤️ Dynamic weather icons
- ❌ Invalid city error handling
- 📱 Responsive UI
- ⚡ Built using Fetch API and Async/Await

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API
